# RR Publishing Site

## Project Overview
RR Publishing is a music publishing company website designed to showcase choral music by Latin American composers, provide information about the company, and allow customers to browse and purchase sheet music. The goal of this project is to build a modular, scalable, and user-friendly frontend using modern web technologies.

This project is built using React and focuses on creating reusable components, responsive design, and an intuitive user experience. The site will eventually include a shopping cart, checkout flow, and user authentication.

## Features
- Browse and search for sheet music by category, composer, or difficulty
- Shopping cart and checkout flow
- Login and registration modals
- Reusable UI components (header, footer, cards, buttons, modals)
- Responsive design for desktop, tablet, and mobile
- Interactive UI with a focus on accessibility and usability

## Tech Stack
- **Frontend:** React, HTML5, CSS3, TypeScript
- **Backend:** Node.js, Express.js, SQL database (for products, users, and orders)
- **Styling:** Tailwind CSS
- **Routing:** React Router
- **Version Control:** Git and GitHub
- **Build Tool:** Vite
